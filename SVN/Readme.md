## Gereksinimler
- Windows (VisualSVN Server için yaygın kullanım)
- Yönetici yetkisi (kurulum için)
- Aynı ağda erişim (web arayüzü / istemci bağlantısı için)

## İndirme Linkleri
- VisualSVN Server: https://www.visualsvn.com/server/download/
- TortoiseSVN: https://tortoisesvn.net/downloads.html

## Hızlı Başlangıç (Özet)
1. VisualSVN Server’ı kur ve yönetim panelini aç.
2. Yeni bir **Repository** oluştur.
3. İstemciye (PC) TortoiseSVN kur.
4. Proje klasörüne sağ tık → **TortoiseSVN → Import**
5. Import sırasında VisualSVN içinden repository URL’sini kopyala-yapıştır.
6. Gerekirse kullanıcı oluştur ve yetki ver.
7. Web arayüzünden repository’i görüntüle.

> Not: Import işlemi klasörün içeriğini yükler; repo adını buna göre seçmek işini kolaylaştırır.
