# SVN Server ve TortoiseSVN ile Kurumsal Sürüm Kontrol Sistemi Kurulumu

Bu repo; **VisualSVN Server** ve **TortoiseSVN** kullanarak kurum içi (lokal ağda) çalışabilen bir **SVN sürüm kontrol sistemi** kurulumunu ve örnek proje yükleme adımlarını içerir.

## İçerik / Kapsam
- VisualSVN Server indirme ve kurulum
- TortoiseSVN indirme ve kurulum
- Repository oluşturma
- Örnek proje import (TortoiseSVN ile)
- Repository URL kopyalama ve kullanıcı oluşturma
- Web arayüzünden repository görüntüleme
